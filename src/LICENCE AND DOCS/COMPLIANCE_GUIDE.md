Compliance Guide

1. Regulatory Standards
- Follows ISO 20022 messaging structure.
- Complies with FATF KYC/AML standards.
- Supports audit trails for each swap.

2. Data Protection
- Encrypted data in transit (TLS 1.3)
- AES-256 encryption for stored data
- Logs anonymized for analytics

3. Licensing Notes
- All deployments must register a License ID with Xere Technologies.
- Country adapters must follow local currency format and FX rate policy.

4. Reporting
- Daily, weekly, and monthly compliance reports auto-generated in `/REPORTS/`.
